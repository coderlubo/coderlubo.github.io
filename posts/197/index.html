Go 语言

2021年7月1日--2021年7月13日

一、语法基础

基础概念

  • 内存地址的编号是一个十六进制的数据

  • 数据都会在内存中进行存储

  • 打印语句: fmt.Println(xxx)

  • 自动推导类型 ==变量名:= 值==

    • 注: 寄存器是临时存储数据的单元

变量命名规范

  • 字母 数字 下划线
  • 必须以字母或下划线开头
  • 严格区分大小写
  • 不能使用关键字
  • 见名知意
  • 命名格式
    • 小驼峰(第一个单词首字母小写) / 大驼峰
    • 单词下划线连接 所有字母小写

基础数据类型

  • 布尔: bool\textcolor{gray}{bool}bool

  • 整型: intint8int16int32int64\textcolor{gray}{int \quad int8 \quad int16 \quad int32 \quad int 64}intint8int16int32int64

    • 注：int 类型会根据操作系统的位数不同而在内存中占的字节也不同

  • 无符号(unsigned)整型: uintuint8uint16uint32uint64\textcolor{gray}{uint \quad uint8 \quad uint16 \quad uint32 \quad uint64}uintuint8uint16uint32uint64

    • 注：出现负数会溢出 由最小值变为最大值

  • 浮点数: float32\textcolor{gray}{float32}float32 单精度 float64\textcolor{gray}{float64}float64 双精度

  • 字符类型: 单引号括起来单个字符

  • 字符串: 双引号括起来的内容

  • 复数: complex128complex64\textcolor{gray}{complex128\quad complex64}complex128complex64

  • 获取字符串长度: len(str)

    • 注: 在go语言中 一个汉字 占三个字节 ⟹\quad \textcolor{orange}\Longrightarrow \quad⟹ 为了和Linux进行统一处理

  • 字符串连接 +

  • fmt 输入输出

    含义
    %% 一个%字面量
    %b 一个二进制整数值(基数为2)，或者是一个(高级的)用科学计数法表示的指数为2的浮点数
    %c 字符型。可以把输入的数字按照ASCII码相应转换为对应的字符
    %d 一个十进制数值(基数为10)
    %f 以标准记数法表示的浮点数或者复数值
    %o 一个以八进制表示的数字(基数为8)
    %p 以十六进制(基数为16)表示的一个值的地址，前缀为0x,字母使用小写的a-f表示
    %q 使用Go语法以及必须时使用转义，以双引号括起来的字符串或者字节切片[]byte，或者是以单引号括起来的数字
    %s 字符串。输出字符串中的字符直至字符串中的空字符（字符串以’\0‘结尾，这个’\0’即空字符）
    %t 以true或者false输出的布尔值
    %T 使用Go语法输出的值的类型
    %x 以十六进制表示的整型值(基数为十六)，数字a-f使用小写表示
    %X 以十六进制表示的整型值(基数为十六)，数字A-F使用小写表示

  • 进制

    • 八进制数以 0\textcolor{gray}00 开头 十六进制以 0x\textcolor{gray}{0x}0x 开头
    • 二进制不能直接表示

  • 常量

    • const  常量名  数据类型  =  值\textcolor{gray}{const\;常量名\;数据类型\;=\;值}const常量名数据类型=值
    • 常量不允许左值赋值 即: 不能被放在等号左侧
    • 定义常量使用大写字母
    • go 语言常量的地址不允许被访问

  • iota 枚举

  • 运算符

    • 算术运算符

      +−∗/%++−−\textcolor {gray} {+ \quad - \quad * \quad / \quad \% \quad ++ \quad --}+−∗/%++−−

      • 注:
        • 取模运算符不能对浮点数使用且除数不能为 0
        • 可以对浮点数进行自增、自减运算 但是不能对常量进行自增、自减运算
        • 自增和自减不能出现在表达式中
          • 避免二义性 在不同操作系统中 运算方式不同 结果可能会产生偏差
        • 自增和自减都只能 后置  不能前置

    • 类型转换

      数据类型(变量名)数据类型(表达式)\textcolor{gray}{数据类型(变量名)\quad 数据类型(表达式)}数据类型(变量名)数据类型(表达式)

    • 赋值运算符

      :=+=−=∗=/=%=\textcolor{gray}{:= \quad += \quad -= \quad *= \quad /= \quad \%=}:=+=−=∗=/=%=

    • 关系运算符

      ><<=>=!===\textcolor{gray}{> \quad < \quad <= \quad >= \quad != \quad ==}><<=>=!===

    • 逻辑运算符

      &&!∣∣\textcolor{gray}{\&\& \quad ! \quad ||}&&!∣∣

    • 其它运算符

      &∗\textcolor{gray}{\& \quad *}&∗

      • &: 取地址运算符
      • *: 取值运算符

    • 单目运算符

      !++−−&\textcolor{gray}{! \quad ++ \quad -- \quad \&}!++−−&

    • 运算符优先级

流程控制

  • 选择结构

    • if 和 switch 的比较
      • 优点
        • if 可以进行区间判断 嵌套使用
        • switch 执行效率高 可以将多个满足相同条件的值放在一起
      • 缺点
        • if 执行效率低
        • switch 不建议嵌套使用

  • 循环结构

函数

  • 函数的定义和使用

  • 注：

    • 在整个项目中 函数名是唯一的 不能重复
    • 函数调用 函数可以多次调用
    • 在函数调用时参数为实际参数 有具体的值 用来给形式参数传递数据

  • 不定参数列表

  • 注：

    • arr 是一个数据的集合
    • len(arr) 可以用来计算数据集合中的数据个数
    • arr[下标] 可以用来找到具体数据的值
    • 下标从 0 开始
    • 不定参使用的数据格式为切片 不是 数组
    • 不能将不定参的名称传递给另一个不定参

  • 函数返回值

  • 函数类型

    • 函数的名字表示函数在代码区的地址

    • type 定义函数类型 为已存在的数据类型起别名

  • 函数作用域

    • 变量要先定义后使用 在函数内部变量名是唯一的
    • 局部变量
      • 在函数内部定义的变量
      • 作用域在函数内部
    • 全局变量
      • 在函数外部定义的变量
      • 在整个项目文件中都可以使用
      • 定义的全局变量名不能和其他文件中的全局变量重名
      • 全局变量名可以和局部变量名重名（就近原则）

  • 匿名函数

  • 递归函数

    • 递归函数 一定 要有出口

工程管理

  • 目录结构 工作区
    • src: 用于以代码包的形式组织并保存 Go 源码文件
    • pkg: 用于存放经 go install 命令构建安装后的代码包
    • bin: 与 pkg 目录类似，在通过 go intall 命令安装完成后，保存由 Go 命令源码文件生成的可执行文件
  • 不同目录下导入方式
    • 要导入的方法首字母要大写
    • 包名必须和所在目录一致
    • GoPath 目录要设置在当前工程目录下

二、 数据格式

数组

  • 指在内存中顺序存储的一系列相同数据类型的元素集合

  • 数组的定义： var 数组名 [元素个数] 数据类型

  • 数组下标从 0 开始，最大为 len(arr)-1

  • 可以直接打印数组

  • 数组的定义和使用

  • 冒泡排序

    • 比较两个相邻的元素，如果满足一定条件就进行数据交换

    • 每执行一周可以确定一个数据是有效的

    • 执行 元素个数-1 次即可确定所有的数据都是有效的

    • 随着执行次数增长，比对次数逐渐减少

  • 插入排序

    • max、index
    • 冒泡排序和插入排序空间优先

  • 随机数

  • 二维数组
  • 数组作为参数传递时的注意事项：
    • 数组作为参数传递时是值传递
    • 形参和实参是不同的存储单元
    • 内存中有两份独立的数组存储不同的数据
    • 在函数调用结束形参单元销毁，不会影响主调函数中实参的值
    • 如果想通过函数计算结果并传递给实参 需要使用数组作为函数的返回值

Map

  • map 的定义和使用

    • Map [key]value Map [键类型]值类型

    • 练习：记录单词中字母出现的次数

  • 注：

    • map 存储的方式不是顺序存储

    • map 的长度是自动扩容的

    • map 中的 key 是唯一的，value 可重复

    • map 中的 key 的类型必须支持 ==、!=

      • 使用 基本类型 即可

    • 在 map 中只能通过 key 找 value

    • 在 map 中如果没有通过 key 找到相应的 value，则打印value类型 的默认值

    • 判断 key 是否存在

  • map 作为参数传递时的注意事项：

    • 地址传递，即: 引用传递
    • 形参和实参指向内存中相同的地址段，修改形参会影响实参的值

接口（多态）

  • 接口就是一种规范与标准

  • 接口的定义与使用

  • 注：

    • 接口是一种数据类型 可以接收满足条件的对象的信息
    • 将对象赋值给接口 对象必须满足接口中方法的声明格式
    • 接口中定义的是方法类型，也即 函数类型
    • 接口是虚的 方法是实的
    • 接口定义规则 方法实现规则
    • 接口定义的规则在方法中必须有对应的实现

  • 多态的实现

    • 接口对象会根据传递过来的对象自动进行分发，调用不同的方法

    • 超集中包含子集的所有方法，子集中不一定包含超集中的所有方法

  • 空接口

  • 类型断言

    • 接口类型不能直接参与运算 需要先通过类型断言判断其类型并获取其值

异常处理

  • error 接口

    • 是一个关于错误处理的标准模式

  • panic 函数

  • 延迟调用 defer

    • defer 用于延迟一个函数（或者当前创建的匿名函数）的执行
    • defer 只能出现在函数的内部
    • defer 的应用场景：
      • 文件操作，先打开文件，执行读写操作，最后关闭文件，为了保证文件的关闭能够正确执行，可以使用 defer
    • 如果在 defer 调用时将函数的参数先放在内存中，是一个独立的空间，不会因为改变值而影响函数执行结果。如果不传递参数，使用外部的变量，如果外部变量修改，则会影响函数的值
